## 🇩🇪 Deutsch
**Automatisches Aufräumen des Papierkorbs für den Linux-Desktop.**

TrashGuard ist ein nützliches Tool, das im Hintergrund die Größe deines Papierkorbs überwacht. Sobald ein von dir festgelegtes Limit überschritten wird, leert der Dämon den Papierkorb automatisch. Entwickelt mit **GTK4** und **Libadwaita** für eine perfekte Integration in GNOME und COSMIC.

### Installation
1. Lade die Datei `trashguard.flatpak` herunter.
2. Installiere sie mit folgendem Befehl im Terminal:
```bash
flatpak install --user trashguard.flatpak
